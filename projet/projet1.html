Projet 1 : Installation de Gitea

Une alternative à GitHub pour gérer le code source localement via Docker.

Résumé du Projet

Le projet consiste à installer Gitea sur un serveur local en utilisant Docker, pour relier les environnements de développement et de production.

Étapes Principales :

  • Configuration des serveurs avec Docker Compose
  • Installation et configuration de Gitea
  • Ajout du projet de développement à Gitea
  • Clonage du projet sur le serveur de production
