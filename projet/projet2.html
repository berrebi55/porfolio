Projet 2 : Serveur Apache et FTPS

Configuration d'Apache pour héberger plusieurs sites et mise en place d'un serveur FTPS sécurisé.

Résumé du Projet

Ce projet a consisté à configurer un serveur Apache avec deux sites web distincts et à mettre en place un serveur FTPS sécurisé à l'aide de ProFTPD. Voici un aperçu des étapes principales :

1. Configuration d'Apache :

  • Installation d'Apache avec sudo apt install apache2
  • Création de deux sites web (SITE1 et SITE2) dans des répertoires séparés
  • Configuration des hôtes virtuels pour chaque site et activation via a2ensite

2. Activation du SSL :

  • Installation d'OpenSSL et génération de certificats SSL
  • Configuration d'Apache pour utiliser le SSL sur le port 443

3. Mise en Place du Serveur FTPS :

  • Installation de ProFTPD et activation du module FTPS
  • Génération des certificats SSL pour ProFTPD
  • Test de la connexion sécurisée avec FileZilla

4. Vérification de la Sécurité :

  • Capture du trafic réseau avec Wireshark pour valider le chiffrement FTPS
